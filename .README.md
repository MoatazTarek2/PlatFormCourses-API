# PlatForm-Courses-API
**PlatForm-Courses-API** is a backend project for an online course platform built with **Node.js, Express, and MongoDB**.  
Instructors can create courses and lessons, while students can book courses and leave reviews.

---

## **Features**
- User registration (Students, Instructors)  
- Login with **JWT authentication**  
- CRUD operations for courses and lessons  
- Course booking for students  
- Add and review lessons and courses  
- Role-based access control (RBAC) for endpoints  
- Rate-limiting on login to prevent brute-force attacks  

---

## **Technologies Used**
- Node.js  
- Express  
- MongoDB + Mongoose  
- JWT Authentication  
- bcrypt for password hashing  
- Joi for data validation  
- express-rate-limit for login protection  

---

## **Requirements**
- Node.js >= 18  
- MongoDB (Atlas or Local)
